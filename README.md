# EagleWay Trucking Application

This is a premium HTML application form for EagleWay Trucking. All form submissions are prepared to be sent to **shaxriyor.z@icloud.com**. 

## Features
- Full Name, Phone, Email, Nationality
- OTR Experience with dynamic payment display based on years entered:
  - 1–2 years: 60 CPM / 26%
  - 3–20 years: 65 CPM / 28%
  - Gross: $8k–$11k weekly
- Weekly miles selection (4000, 4500, 5000)
- Working cycle options (4/4, 3/3, 5–6 weeks on/off)
- ELD field (agreed, non-editable)
- Endorsements: TWIC, Hazmat, Tanker
- CDL Front & Back upload
- Medical card upload
- Trailer preference (Dry Van, Reefer, Both)
- Payment / Rates displayed in a separate, highlighted frame
- Escrow field (fully refundable, non-editable)
- Additional notes section

## Instructions
1. Replace `logo.png` with the actual EagleWay logo in the project root.
2. Upload all files to GitHub repository.
3. Users can fill out the form and submit via their email client.
4. Escrow and ELD fields are not editable by users.
5. Payment updates dynamically based on OTR experience entered.
